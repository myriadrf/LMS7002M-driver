This is software to control Lime Microsystems LMS7 and Stream boards

Detailed instructions how to compile this software is given in 
LMS7Suite_compilation_guide.pdf document.

NOTICE for Linux users:
	To be able to use USB devices, LMS7Suite needs to be executed with
	administrative privileges. Or the USB device permissions have to be changed
	to allow non-root users to acces them.
	USB-3.0 connection is not supported.
